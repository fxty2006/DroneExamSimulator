# 🚁 ドローン国家資格 学科試験シミュレーター

ドローン国家資格（一等・二等無人航空機操縦士）の学科試験対策アプリです。
Google Gemini API を活用し、公式の教則PDFの内容に基づいた問題を**無限に生成**することができます。

![license](https://img.shields.io/github/license/fxty2006/DroneExamSimulator)
![python](https://img.shields.io/badge/python-3.10%20%7C%203.11%20%7C%203.12-blue)

## ✨ 主な機能

*   **🤖 AI問題生成**
    *   国土交通省の公式「教則PDF (rules.pdf)」をAIに読み込ませ、最新の法令に対応した三択問題を自動生成します。
    *   問題のストックが尽きることがありません。
*   **📚 模擬試験モード**
    *   🔰 **二等** (30分) / 👑 **一等** (75分)
    *   解説付きの「練習モード」と、時間を計って一気に解く「本番モード」を搭載。
*   **📊 弱点分析 & 復習**
    *   試験終了後、章ごとの正解率をグラフで表示。
    *   間違えた問題だけをピックアップして再テストが可能。
*   **🚀 簡単セットアップ (ポータブル)**
    *   必要なライブラリを同梱しているため、Python環境の構築作業は不要です。
    *   解凍してクリックするだけで動作します。

## 🔑 APIキーと制限について

### 1. 試験シミュレーター (問題を解く)
**APIキー不要・完全オフライン** で動作します。
本パッケージにはサンプル問題データが同梱されているため、ダウンロードしてすぐに試験を試すことができます。

### 2. 問題ジェネレーター (問題を作る)
新しい問題をAIに作成させたい場合のみ、無料の Google Gemini APIキー が必要です。

**【推奨設定と作成上限】**
*   精度の高い問題を生成するため、本ツールでは **「Proモデル (gemini-2.5-pro 等)」** の使用を推奨しています。
*   Proモデルを無料枠で使用する場合、API制限により **1日に作成できるのは「合計 約120問」程度** が目安です。
    *   ※制限を超えても、翌日になればまた作成できるようになります。

## 📥 ダウンロードと使い方

### ✅ 一般ユーザー向け (推奨)
**[Releases ページ](https://github.com/fxty2006/DroneExamSimulator/releases)** から最新のZIPファイル (`DroneExam_v0.x.x.zip`) をダウンロードしてください。

1.  ダウンロードしたZIPを**必ず「展開（解凍）」**する。
2.  フォルダの中にある `start.bat` をダブルクリックして起動。
3.  メニューの **`[2] 試験を始める`** を選べばスタート！
    *   ※この方法は **完全オフライン** で動作します (Python 3.10/3.11/3.12 対応)。

### 🛠 開発者向け (Source Code)
このリポジトリを Clone または Download して使用する場合、`libs` フォルダが含まれていないため、起動時に **インターネット経由でライブラリが自動インストール** されます。

1.  コードをダウンロードして解凍。
2.  `start.bat` を実行（※インターネット接続必須）。
3.  自動的に環境構築が行われ、アプリが起動します。

## ⚠️ 注意事項

*   AI生成のため、稀に誤った問題文や解説が含まれる可能性があります。必ず公式の教則と照らし合わせて学習してください。
*   本アプリは「公開テスト版」です。予期せぬ不具合が含まれる可能性があります。
