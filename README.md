# 🚁 ドローン国家資格 学科試験シミュレーター (v0.1.0)

ドローン国家資格（一等・二等）の学科試験対策アプリです。
Google Gemini API を活用し、教則PDFの内容に基づいた問題を**無限に生成**することができます。

今回は**「公開テスト版（人柱募集）」**としてのリリースです。ぜひ試していただき、フィードバックを頂けると嬉しいです！

## ✨ 主な機能
*   **AI問題生成**: 教則PDF（rules.pdf）を読み込み、最新の法令に対応した三択問題を自動生成します。
*   **模擬試験モード**:
    *   🔰 **二等** (30分) / 👑 **一等** (75分)
    *   解説付きの「練習モード」と、一気に解く「本番モード」を搭載。
*   **弱点分析 & 復習**: 間違えた問題の傾向をグラフ化し、苦手な問題だけを再テストできます。
*   **簡単セットアップ**: 必要なライブラリを同梱しているため、ネット環境がなくてもセットアップ可能です。

## 🔑 APIキーについて（重要）
**試験シミュレーター（問題を解く機能）は、APIキー不要・完全オフラインで動作します！**
本パッケージにはサンプル問題データが同梱されているため、**ダウンロードしてすぐに試験を試すことができます。**

※ Google Gemini APIキー（無料）が必要になるのは、**「新しい問題をAIに追加作成させたい時」のみ**です。

## 📦 使い方
1.  下の **Assets** から `DroneExam_v0.1.0.zip` (※ZIPファイル名) をダウンロードしてください。
2.  ダウンロードしたZIPを**必ず「展開（解凍）」**してください。
3.  フォルダの中にある `start.bat` をダブルクリックして起動します。
4.  メニューの **`[2] 試験を始める`** を選べば、すぐにスタートできます。

## ⚠️ 注意事項
*   AI生成のため、稀に誤った問題文や解説が含まれる可能性があります。必ず公式の教則と照らし合わせて学習してください。
*   テスト版のため、予期せぬ不具合が含まれる可能性があります。
